Luminus

Aplicação mobile realizada com React Native durante as aulas de desenvolvimento mobile.

GITHUB

Visão geral do projeto

Status: Em desenvolvimento.

Repositório destinado ao projeto da matéria de Programação para Dispositivos Móveis I, ministrada pelo professor Gerson Penha, durante o 4º semestre do curso de Desenvolvimento de Software Multiplataforma, na FATEC de São José dos Campos - Jessen Vidal.

Na matéria, o objetivo desse projeto era que cada aluno escolhesse um tema para desenvolver um aplicativo móvel. Escolhi fazer uma aplicação para auxiliar profissionais da saúde e pacientes/familiares de pessoas que necessitam cuidados à domicílio, buscando simplificar e organizar a obtenção de registros das informações de pacientes, seus casos e seus cuidados. Mesmo tendo sido aprovada na matéria, o desenvolvimento e aperfeiçoamento desse projeto seguem ativos.
